# sales_data_analysis_python
Project Overview

This project focuses on transforming raw sales data into actionable business insights using Python and the Pandas library. It was completed as part of the Week 2 Internship Project to demonstrate skills in data manipulation, revenue calculation, and performance reporting.

Key Tasks Completed

Dataset Creation: Generated a structured dataset containing over 50 sales records, including order details, product categories, quantities, and pricing.
Revenue Calculation: Successfully calculated the total sales amount for every individual record.
Performance Analysis: Conducted product-wise and category-wise sales analysis to identify top-performing areas.
Time-Series Analysis: Performed monthly sales trends analysis using order dates.
Data Export: Exported the final processed data into a clean CSV format named week2_sales_analysis_output.csv.

Key Business Insights

Top Category: The Electronics category emerged as the primary driver of revenue.
High-Value Product: The Laptop was the highest-priced item, significantly impacting total revenue.
Volume Leader: Stationery items like Pen Packs and Notebooks showed the highest quantity sold, indicating high frequency of purchase.

Tools Used

Language: Python
Environment: Google Colab
Library: Pandas
